## Software de Grado de Esencialidad de Sistemas y Equipos Navales - GRES 2.0
Software de Grado de Esencialidad de Sistemas y Equipos Navales - GRES es un software especializado diseñado para implementar la Metodología GRES en Cotecmar. Esta herramienta facilita y optimiza el proceso de análisis y clasificación de sistemas y equipos, permitiendo una gestión eficiente de la información relacionada con la esencialidad de misión. El software ofrece las siguientes características principales:

- Sistema integrado de gestión de datos para análisis de equipos y sistemas
- Motor de clasificación basado en la Metodología GRES
- Interfaz intuitiva para el ingreso y consulta de información
- Sistema de reportes
- Gestión de usuarios y roles
- Trazabilidad completa de las decisiones de clasificación
